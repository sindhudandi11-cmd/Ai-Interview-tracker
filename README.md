🎯 Interview Prep Tracker (AI-Powered)

An AI-powered full-stack web application that helps users track job applications, save interview questions, take notes, and generate personalized interview questions from job descriptions using AI.

✨ Features

📋 Job Applications Tracker
Add, update, and delete job applications
Track status: Applied / Interview / Rejected / Accepted
Organized application dashboard

❓ Interview Questions Manager
Save interview questions by company
Add personal notes and solutions
View all saved questions in one place

📝 Notes System
Save personal interview prep notes
Timestamped entries for revision

🤖 AI Interview Question Generator
Paste a job description and generate:
Role analysis (tech stack, experience level, responsibilities)
5 tailored interview questions
Or generate a question from role name + company
Uses OpenRouter AI models
